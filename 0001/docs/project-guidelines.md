# Project: Shopping cart service

## General guidelines
- Put all packages and classes under the package "com.example.shopping_cart".
- Put the @SpringBootApplication class in the package "com.example.shopping_cart".
- Put database schema in the directory "/data".
- Ignore the files and directories: @README.md, .git.
- Do not use brace expansion ({}) in shell commands such as mkdir.
- When creating multiple directories, list each one explicitly by using mkdir -p /app/dto /app/mapper instead of mkdir -p /app/{dto,mapper}.
